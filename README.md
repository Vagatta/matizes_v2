# Pastelería Mat - Sitio Web Astro

Proyecto Astro para la pastelería fina Mat!zes, convertido desde HTML/CSS a componentes Astro.

## 🚀 Instalación y Ejecución

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Iniciar servidor de desarrollo:**
   ```bash
   npm run dev
   ```

3. **Construir para producción:**
   ```bash
   npm run build
   ```

4. **Previsualizar producción:**
   ```bash
   npm run preview
   ```

## 📁 Estructura del Proyecto

```
src/
├── layouts/
│   └── Layout.astro          # Layout principal con estilos globales
├── pages/
│   └── index.astro           # Página principal con todas las secciones
```

## 🎨 Características

- **Astro Framework**: Sitio estático optimizado
- **Tailwind CSS**: Estilos mediante CDN
- **Font Awesome**: Iconos integrados
- **Google Fonts**: Tipografía Cormorant Garamond y Jost
- **Diseño Responsivo**: Mobile-first approach
- **Animaciones**: Transiciones suaves y efectos hover

## 📄 Secciones del Sitio

1. **Header**: Navegación fija con logo y carrito
2. **Hero**: Sección principal con imagen de fondo
3. **Brand Statement**: Filosofía y estadísticas
4. **Featured Products**: Productos destacados
5. **Categories**: Galería de categorías
6. **Special Order**: Pedidos personalizados
7. **Testimonials**: Reseñas de clientes
8. **Newsletter**: Suscripción a novedades
9. **Footer**: Información de contacto y redes sociales

## 🎨 Colores y Estilos

- **Primary**: Purple (#5c1a4a)
- **Secondary**: Gold (#d0c717)
- **Background**: Cream (#faf8f4)
- **Typography**: Cormorant Garamond (serif), Jost (sans-serif)

## 📱 Responsive Design

El sitio está optimizado para:
- Móvil: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## 🔧 Configuración

- **Astro Config**: Configurado para salida estática
- **Tailwind**: Configurado con colores personalizados y tipografía
- **Componentes**: Estructura modular con layout base
